# Сайт фотографа (photographer-site)

Одностраничный сайт-портфолио с отдельными разделами для разных типов съёмок:

- Женский портрет
- Семейная фотосессия
- Фуд-съёмка
- Продвижение бизнеса

Сделан на **Next.js 14 (App Router)** и **Tailwind CSS** с упором на визуал и удобство просмотра портфолио.

## 🚀 Запуск проекта локально

```bash
cd photographer-site
npm install
npm run dev
```

Откройте в браузере: **http://localhost:3000**

## 📁 Основные страницы

- `/` — главная, геро-блок, об услугах, оригинальный блок «Мое портфолио», форма заявки
- `/portfolio/women-portrait` — портфолио «Женский портрет»
- `/portfolio/family` — портфолио «Семейная фотосессия»
- `/portfolio/food` — портфолио «Фуд-съёмка»
- `/portfolio/business` — портфолио «Продвижение бизнеса»

## 🎨 Настройка под себя

- Замените заглушки галерей на реальные фото (можно подключить локальные изображения или CDN)
- Обновите контакты в `app/layout.tsx` и на главной странице
- При необходимости доработайте тексты и структуру секций
